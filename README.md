# Hoshi no Meishiki

日本向け占い・命式ツール `星の命式` の MVP リポジトリです。

## Preview

Static site files live in `site/`.

Open locally:

```bash
python3 -m http.server 4173 --directory site
```

Then visit:

```text
http://127.0.0.1:4173/
```

## GitHub Pages

Publish the `site/` folder with GitHub Pages, or deploy it with any static hosting provider.
